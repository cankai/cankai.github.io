今日头条面试记

2017-09-13 22:20:00
By Chen Kai

今日头条的面试分为两部分，一部分是算法与编码，一部分就是闲聊技术了，算法与编码那部分感觉达得还OK，题目记录下来。

一、链表相加

有两个链表，表示两个数字，数字高位位于链表前面，求两个链表的说代表的数字的和，输出和所代表的链表，例如：链表1（1->2->3->4->5->6->7->8),链表2（7->8->9)，那么输出的链表应该是(1->2->3->4->6->4->6->7);因为链表的操作是从链表头到链表尾循环，而数字的加分是低位到高位顺序相加，因此可以先将链表翻转，然后相加。

typedef struct Node Node;
struct Node{
    int val;
    Node* next;
    Node(int v, Node* n = NULL)
    {
        val = v;
    }
}

二、数组区间求值

给出一个数组，代表在一个二维空间里面的山的高度，两个山峰之间是山谷，山谷中可以存雨水，求当前数组中，可以存多少水量。 定义：山峰：两边都比当前数字小。 所以只要找到两个山峰，求两个山峰之间的水量就好了；例外情况：当左边山峰已经固定，如果右边出现一个山峰，比左边的山峰矮，那么这个山峰不为山谷的左边，除非左边已经没有更高的山峰。笔试的时候是在电脑上写题目，两个题目四十分钟，这一题写出来有瑕疵。
